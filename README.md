# Guess the Number

Projeto de prática de front-end que consiste em um **jogo de adivinhação de números**, desenvolvido com HTML, CSS e JavaScript.

## 📋 Sobre o projeto

Um jogo simples onde o usuário deve adivinhar um número aleatório gerado pelo sistema, dentro do intervalo de 1 a 100, possuindo um número limitado de tentativas para acertar.

## 🖼️ Preview

A página exibe:

- **Título:** "Guess a number from 1 to 100"
- **Campo de input:** para o usuário digitar seu palpite
- **Botão:** "Check" para validar a tentativa
- **Contador de tentativas:** "You have 10 chances"
- **Fundo:** gradiente em tons de azul
